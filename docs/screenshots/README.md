# Screenshot Uploads

Place screenshot files for the lab here before committing to GitHub.

Recommended filenames:
- `health-local.png`
- `classify-image-demo-local.png`
- `docker-images.png`
- `docker-containers.png`
- `docker-logs.png`
- `swagger-docs-container.png`
- `classify-image-demo-container.png`

Then run:

```bash
git add docs/screenshots/*
git commit -m "Add lab screenshots"
git push
```
